# EdTech-User-Engagement-Analysis

## Overview
This project performs Exploratory Data Analysis (EDA) on an EdTech platform user engagement dataset using Python and Sweetviz.

## Objectives
- Analyze user engagement patterns
- Explore platform usage behavior
- Generate automated EDA reports
- Identify trends and insights from the dataset

## Tools Used
- Python
- Pandas
- NumPy
- Sweetviz
- Jupyter Notebook

## Project Files
- EdTech_Platform_User_Engagement_Analysis.ipynb
- EdTech_User_Engagement_Report.html
- Dataset (.csv)

## Output
The project includes a Sweetviz report that provides automated visualizations and statistical insights into user engagement behavior.
